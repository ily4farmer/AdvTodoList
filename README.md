# Advanced Todo-List
##### GhPages - https://ily4farmer.github.io/AdvTodoList/
---
## Используемые технологии
* React 
* React-Hooks ( useReducer, useState, useEffect, useContext ) 
* LocalStorage
* Sass
---
## Функционал
* Создание своих задач
* При клике на задачи создание подзадач 
* Отметить выполнение подзадачи
* Удалить задачу
* Сохранение всех задач и подзадач после перезагрузки страницы
---
## Установка
1. git clone https://github.com/ily4farmer/AdvTodoList.git
2. npm i
3. npm start

##### Build версия проекта - npm run build
##### Деплой на GhPages - npm run deploy (Если вы хотите сразу задеплоить проект на GhPages не используйте npm run build, т.к перед деплоем на хостинг в любом случае будет создоваться build версия проекта)
